# Online Classes App

## How to Run backend?
- You have to have Docker installed
- Open terminal
- Change the directory to "backend" - (cd backend)
- run "docker-compose run --build"
- Wait till it says "INFO:     Uvicorn running on http://0.0.0.0:8000"
